planningpoker
=============

Application
=============
http://githubplanning.com/

Developing
=============

Install node http://nodejs.org/ and npm https://www.npmjs.org/
Instal dep 
```shell 
npm install 
```
Install Grunt
```shell 
npm install -g grunt-cli
```
Enable watch task 
```shell 
grunt
```
Start process 
```shell 
sudo node app.js 
```
Or install supervisor. he works even better
```shell 
npm install supervisor -g
```
Then run
```shell 
sudo supervisor app.js
```
Open http://127.0.0.1/
