Developing
=============

Install node http://nodejs.org/
Iinstall npm https://www.npmjs.org/
Instal dep 
```shell 
npm install 
```
Enable watch task 
```shell 
grunt
```
Start process 
```shell 
sudo node app.js 
```
or install supervisor. he works even better
```shell 
npm install supervisor -g
```
then run
```shell 
sudo supervisor app.js
```
open http://127.0.0.1/
