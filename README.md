Install
=============
1. install [node]
[node]: http://nodejs.org/
2. install [npm]
[npm]: https://www.npmjs.org/
3. make command
```bash
npm install
```

4. start process

```bash
node app.js
```
5. Go on browser by link http://127.0.0.1:3000
